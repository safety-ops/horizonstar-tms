  <files>assets/css/base.css</files>
  <action>
Make these targeted edits to existing classes in base.css:

**Cards (.card):**
- Change `border-radius: 12px` to `border-radius: var(--radius)` (will resolve to 8px from updated tokens)
- If there is a `.card:hover` with `transform`, remove the transform property (keep background/shadow changes if any)
- Do NOT add/remove the `.card::after` pseudo-element here -- that is in the index.html inline block, handled in Plan 03

**Hero cards (.hero-card.green, .hero-card.blue, .hero-card.amber, .hero-card.red, .hero-card.purple):**
- Remove any `background: linear-gradient(...)` declarations
- Replace with `background: var(--bg-card)`
- Keep `border-left-color` accent (that is the only color accent for hero cards)

**Stat cards (.stat-card):**
- Change any `.stat-value` with `font-weight: 800` or `font-weight: 700` to `font-weight: 600`
- Keep `font-size` as-is

**Status badges (.status-badge):**
- Change `border-radius: 12px` (or any pill radius) to `border-radius: var(--radius-sm)` (6px)
- Change `font-weight: 600` to `font-weight: 500`
- Dim backgrounds will auto-update via the token change (12% to 8%)

**Tables (.data-table):**
- Remove zebra striping: find `tbody tr:nth-child(even)` background rule and remove it or set `background: transparent`
- Remove `backdrop-filter: blur(8px)` from thead if present
- Change th `font-weight: 600` to `font-weight: 500`
- Ensure th has `background: var(--bg-tertiary)` (opaque, no blur)

**Filter chips (.filter-chip):**
- Change `border-radius: 20px` to `border-radius: var(--radius-sm)` (6px, not pill-shaped)

**Page headers (.page-header h1, .page-header h2):**
- Change `font-weight: 700` to `font-weight: 600` if present

**Headings (h1-h6):**
- Verify `font-weight: 600` (should already be correct in base.css)
- Add `letter-spacing: -0.01em` for tighter tracking if not present

Do NOT add !important. Do NOT remove any class definitions, only modify property values. Do NOT touch .dark-theme rules.
